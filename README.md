# Личный сайт на Hugo

Минимальный стартовый проект для сайта "Обо мне + блог".

## Что уже есть
- Главная страница и страница `Обо мне`
- Блог из Markdown в структуре разделов
- Светлая/тёмная тема
- Переключатель подложки чтения (Paper/Sepia/Mist/Night)
- Типографский стиль с красивыми шрифтами

## Структура контента
- Разделы блога: `content/blog/<razdel>/_index.md`
- Статьи раздела: `content/blog/<razdel>/<slug>.md`

Пример:
- `content/blog/dev/_index.md`
- `content/blog/dev/moya-statya.md`

Так структура в репозитории совпадает со структурой на сайте.

## Запуск
1. Установи Hugo (extended).
2. В корне проекта:
   - `hugo server -D`
3. Открой `http://localhost:1313`

## Публикация
- Локальная сборка: `hugo`
- Готовый сайт будет в папке `public/`
